      2. Airport Table

        3. Airport table: This table stores information about airports such as airport code, name, location, and possibly other relevant details. It helps in managing the list of airports available for departure and arrival locations when users search for flights.


      3. Flight Table

        4. Flight table: This table holds data about individual flights, including flight number, departure airport, arrival airport, departure time, arrival time, duration, and possibly other flight-related details. It is crucial for users to search for available flights and make reservations.


      4. Reservation Table

        5. Reservation table: This table manages reservations made by users for specific flights. It includes data such as reservation ID, user ID, flight ID, seat number, reservation status, and possibly other reservation-related information. It helps in tracking and managing user bookings.


      5. Payment Table

        6. Payment table: This table contains information related to payment transactions made by users for booking flights. It includes data such as payment ID, amount, payment method, transaction status, and possibly other details necessary for payment processing and tracking.


      6. User Table

      User table: This table stores information about users who interact with the airline booking system. It includes data such as user ID, name, email, phone number, username, password (hashed), and possibly other user-related details. It facilitates user authentication, account management, and personalized services.
